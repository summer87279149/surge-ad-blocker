# Surge 去广告模块合集

为多个热门 App 提供可直接导入 Surge 的去广告模块。复制对应的 Raw 链接，在 Surge → Modules → Remote Module 中添加即可使用。

## 已提供的模块

| App | 说明 | Surge 远程链接 |
| --- | --- | --- |
| 哔哩哔哩 | 屏蔽启动页、信息流广告并阻断商业化上报 | `https://raw.githubusercontent.com/summer87279149/surge-ad-blocker/main/apps/bilibili/BiliAdBlock.sgmodule` |
| 高德地图 | 拦截高德广告素材下载、内部日志与第三方曝光跟踪 | `https://raw.githubusercontent.com/summer87279149/surge-ad-blocker/main/apps/amap/AmapAdBlock.sgmodule` |
| QQ 音乐 | 禁止腾讯音乐广告配置、素材拉取与广点通曝光上报 | `https://raw.githubusercontent.com/summer87279149/surge-ad-blocker/main/apps/qqmusic/QQMusicAdBlock.sgmodule` |
| 米家（小米） | 去除米家首页推荐/商城推广并阻断埋点上报 | `https://raw.githubusercontent.com/summer87279149/surge-ad-blocker/main/apps/xiaomi/XiaomiAdBlock.sgmodule` |

## 使用须知

- 这些模块仅供个人学习、研究网络请求结构之用，不代表或鼓励任何违规行为。
- 请确保遵守当地法律以及目标 App、服务的用户协议。任何使用造成的后果由使用者自行承担。

复制链接 → Surge → Modules → Add → Remote Module → 粘贴 → Enable，即可生效。若需新增其他 App，请关注仓库更新。
